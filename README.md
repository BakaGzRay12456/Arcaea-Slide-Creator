# Arcaea-Slide-Creator

## 简介
Arcaea-Slide-Creator 是一个用于生成类似 Arcaea 中 Slide（滑键）效果的工具。其核心原理是利用**浮点 Hold**（即持续时间极短的 Hold 音符）进行密集排列，模拟出连续的滑动轨迹，从而在谱面编辑器中创建视觉上类似于 Slide 的音符序列。

## 原理
在 Arcaea 中，Slide 通常由一系列连续的片段组成，而 Hold 音符可以设定任意的起始时间和结束时间。通过将大量 Hold 音符的持续时间设置为极小的浮点值（例如 0.001 秒），并在时间轴上以微小间隔排列，可以近似模拟出 Slide 的连续轨迹。这种方法的优势在于：
- 无需依赖特定的谱面格式支持；
- 可以在现有谱面编辑器中直接使用；
- 可自由调整滑动的形状和速度。

## 功能特点
- 生成多种预设滑动轨迹（直线、曲线、折线等）；
- 支持自定义轨迹控制点；
- 输出为可直接导入谱面编辑器的格式；
- 轻量级，无外部依赖。
